# Shot Gun Frog
正在开发的一款防守游戏……

## ⭐ About the Project
主角是一个持枪的青蛙， 不断的发射散弹生存下去……

## 🕹️ Controls

The game is played with the keyboard and mouse:

|Action         |Key               |
|:-------------:|:----------------:|
|Move Forward   |W                 |
|Move Right     |A                 |
|Move Backward  |S                 |
|Move Left      |D                 |
|Move Camera    |Mouse             |
|Shoot          |Left Mouse Button |
|Focus          |Right Mouse Button|
|Menu           |Escape            |
